# 🐉 Chú Bé Rồng Private Server — Termux Auto Setup

## One-liner Termux
```bash
curl -fsSL https://raw.githubusercontent.com/akah3674-glitch/rem5/main/setup.sh | bash
```

## Yêu cầu
- Termux (Android)  
- ~2GB dung lượng trống (RAR tự xóa sau cài)
- Kết nối internet ổn định (~629MB download)

## Hướng dẫn
1. Chạy lệnh one-liner trên Termux  
2. Đợi setup (~15-20 phút tùy mạng)  
3. Server tự khởi động sau setup  
4. Cài APK mod từ `~/Downloads/ChuBeRong_mod.apk`  
5. Trong game → **Nhập IP và PORT** → nhập IP hiển thị + port `14445`

## Lần sau
Chạy lại one-liner → vào menu quản lý server (không tải lại RAR).
